# Houzeo Frontend Assignment

## Tech Stack
- Vue 3 (Vite)
- Tailwind CSS
- Leaflet (OpenStreetMap)

## Features
- Responsive header and filter bar
- Interactive US map
- Property cards with images
- Hover animations and pulse effect
- Mobile responsive (iPhone 14 Pro Max)

## How to Run

1. Install dependencies:
   npm install

2. Run project:
   npm run dev

3. Open:
   http://localhost:5173

## Notes
- UI focused as per assignment
- No backend functionality
